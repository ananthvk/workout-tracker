# workout-tracker


# How to run?
First clone the application,
```
$ git clone https://github.com/ananthvk/workout-tracker
$ cd workout-tracker
```

## To run the application

```
$ docker compose up
```

View the application at [http://localhost:3000](http://localhost:3000)

## To run the tests

```
$ docker compose up -d db
$ pnpm test
```
