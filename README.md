# workout-tracker
## Overview
The primary purpose of this application is to track and derive insights into workout data. After a workout, people do not usually record or track details about their workout, such as weights used, reps performed, or total workout time since it is time consuming to do so. These details are crucial to record progress. This app can help to provide details and suggestions to enable the user to reach their goal.

## Homepage
[https://github.com/ananthvk/workout-tracker](https://github.com/ananthvk/workout-tracker)

## Features
- Web based app

## Technical
- Backend: Nodejs (fastify)
- Fronted: Svelte
- Deployment: Docker
- Database: PostgreSQL
- CSS: TailwindCSS

## Getting started
### Pre-requisities
Install [Docker](https://docs.docker.com/engine/install/) and [docker-compose](https://docs.docker.com/compose/migrate/) 
### Installation
Clone the repository
```
$ git clone https://github.com/ananthvk/workout-tracker/
```
Run the application with
```
$ docker compose up
```
View the application at [http://localhost:3001](http://localhost:3001)
